锦江都城酒店

锦江都城品牌作为锦江卢浮亚洲旗下专注于传播多元文化体验的文化产业型个性化连锁酒店品牌，超过138家的酒店遍布中国80多个城市。我们致力于搭建东西方文化交流的桥梁，演绎中华传统文化，展现世界多元文化，促进世界文化交流。

品牌介绍

锦江都城品牌依托于锦江卢浮亚洲中法团队的国际化管理，专注于传播多元文化体验的文化产业型酒店品牌，致力于搭建东西方文化交流的桥梁，演绎中华传统文化，展现世界多元文化，让充满好奇心、追求全新体验的全球游客享受一种别致的文旅待客之道。

集团背景

锦江国际集团是中国规模最大的综合性旅游企业集团之一，是全球五大顶级酒店集团之一，注册资本20亿元，总资产600亿元，设有酒店、旅游、客运物流、地产、实业、金融六个事业部。

锦江国际集团是亚洲第一、全球第五的酒店集团，全球管理将近八千家酒店，拥有1.5亿会员。2017年上海锦江卢浮亚洲酒店管理有限公司正式成立，是国企旗下覆盖一带一路国家及地区的国际化的专业酒店管理公司。
